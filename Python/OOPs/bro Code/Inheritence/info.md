# Inheritance

## Multiple inheritance

A class can inherit from more than one parent classes </br>
</br>

_Syntax: ```class <class_name>(parent_class-1, parent_class-2, and so on...)```_ </br>
`class C(A, B)`
</br>

## Multilevel inheritance

A class can inherit from a parent class which would be a child class to another parent class
</br>

_Syntax:`C(B)<-B(A)<-A`_
